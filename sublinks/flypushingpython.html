Learning Genetics with Python

A stock is group of flies. To get a desired genotype, one has to cross flies with a set of rules. I wrote a python script in my spare time to replicate the elements of a cross to generate a stable stock (preserving the genotype of the fly). Writing this script helped me understand the complexity behind a simplest cross. It is possible that fundamentals of genetics can be easier to learn with programming logic.  

 

Perspective:

Drosophila is a cool model organisms that is studied in developmental biology. Fly genetics is extremely advanced with a history of hundred years,(Thomas Hunt morgan.) The two major tasks for any fly pusher (handler of the flies) are taking care of the stocks and learning to make crosses as well a complex array of other tasks which i wont go into detail. Both of these are extremely time-consuming and also error prone.

 

Background:

Drosophila have 4 chromosomes. They are known in the order X/Y ; 2R/2L; 3R/3L; 4 for a male and X/X ; 2R/2L; 3R/3L; 4 for a female. Since 4th chromosome is too small, it is usually not targeted. The majority of the genetics done are targeting the first 3 chromosomes. The elegant simplicity of using Drosophila for genetics comes with chromosomes called balancers. These are chromosomes that underwent mutagenesis so much that they donâ€™t recombine anymore (There is still a extremely tiny likelihood that this might happen though).There are several standard balancers available for a typical fly pusher for all the chromosomes. The existence of this chromosome in the fly sometimes comes with a marker in the adult, larvae or pupae fly. This makes easier to select for individuals that are the correct genotype. Balancer chromosomes have many mutations that make the chromosome almost entirely nonfunctional. Having a balancer in the system allows us to only think of one chromosome at a time and not worry about recombining with the other one. In the condition when two balancers come together on the same chromosome, the individual usually dies in the embryonic stage. The tradeoff is the health of the flies as they tend to be much weaker with more number of balancers than a wild type fly.

Example: CyO is one such type of balancer on the second chromosome. It has a dominant marker that causes the adult wings to curl upwards depending upon the temperature. For an excellent reference, check Atlas of drosophila morphology.

Balancers are predominantly of two types. One is a weak balancer effect. Second is a strong balancer. The weak one is used typically used for its marker properties. Since it can just be a normal/partially functional chromosome or not so much mutated chromosome with a marker, the likelihood of recombination in case of pairing with a normal chromosome is much higher. Stronger balancers will preserve whatever is on the other chromosome and can usually considered as the gold standard for balancing.

 

A stable stock has one or more following characteristics. Some of these are easier to quantify than others.

Does not die because of bad health.

Chances of recombination between chromosome arms is negligible.

Can be homozygous on one or both arms.

Can have a strong balancer with the other as a wild type chromosome or a chromosome of your preference on one or more than one chromosomes

Can have a strong balancer with the other as a marker chromosome on one or more than one chromosomes

Can be propagated many times without threat of recombination

Should not have the following:

Two different genotype chromosomes. Causes higher chance of recombination

Genotype chromosome with a marker. Much lower chance, but still there is a likelihood of recombination depending upon the weak balancing effect.

It is still possible to use a non-stable stock for your experiments. Just discard them after you finish doing the experiment. It is not wise to propagate such stocks as due to the threat of recombination. This means it is not always easy to say what is the genotype of the next generation? This can considered to be an unclean way of doing a cross. Recombination does not happen in males, so it is possible to imagine a scenario where you can use the males to pass the genes to a next generation without recombination. But that will of course will use too much energy.

 

Script:

So far it is restricted to only the 2nd and 3rd chromosomes. It creates a score of how stable is a stock. Anything which can cause recombination to occur gets a lower score.

The inputs of the script, make sure these files are just provided in the directory are

stocks.csv - contains a global list of the stock collection with their genotypes.

markers.csv - Contains list of common markers but not balancer chromosomes. It is just a code for the marker.

balancers.csv - contains a list of balancers, it is just to tell the script the code for a particular balancer

crosses.csv - this is typically the only input file that you need to touch. The above three are kind of initialization values. Tell the software which genotype you would prefer if you cross a particular two genotypes. Alternatively. Just a target genotype should suffice. The software should be able to search for the transgenes, the balancers and give you a best possible parents to choose from. This is not implemented as of now.

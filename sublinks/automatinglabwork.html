Automating lab work

 

The two major tasks for any handler of the flies is taking care of the stocks and learning to make crosses as well a complex array of other tasks which i wont go into detail. Both of these are extremely time-consuming, repetitive and error prone. It is possible to bring in an array of automated methods to do most of the work so that the researchers can spend their time actually thinking about the experimental setup rather than worrying about the flies. This increases reproducibility as contamination one of the big issues.

An Example of a lab automation, fly catching system.

Modules can built around such a robot so that fly handling becomes service than a mandatory workload for a human. It can be even be viable alternative to stock generation if equipped with an automated injection module, such a module will need submodules such as egg collectors, larvae pickers.

This task by itself is not easy. Many of the components for the automation require a combination of image processing, robotics, automated climate control hoods etc. But the whole system can be built modular removing elements which are not needed. It can be made so that a certain number of individual flies can be sampled at every generation for PCR/Sequencing to check whether the genotype is still preserved.
